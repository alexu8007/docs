# /paragon - AI Code Review Page

## Visual Components

### Page Header
**What you see**: Top section with branding and status
- **Title**: "Paragon" in large text
- **Subtitle**: "AI-powered code review for your pull requests"
- **Status indicator**: Green dot "Online" or red dot "Offline"
- **Stats badges** (right side):
  - "Reviews today: 12"
  - "Avg response: 45s"

### PR Review Queue Section
**What you see**: Main content area with PR list

#### Section Header
- **Title**: "Pull Requests Ready for Review"
- **Count badge**: Shows number (e.g., "8 PRs")
- **Refresh button**: Circular arrow icon
- **Filter dropdown**: "All", "Urgent", "Ready", "Draft"

#### PR Cards Grid
**What you see**: Cards arranged in a grid (2-3 per row)

Each PR card contains:
- **Repository name**: Top left with repo icon
- **PR title**: Bold text, main focus
- **PR number**: "#123" in gray
- **Author**: Small avatar + username
- **Status badges**:
  - "Urgent" (red) - Requested reviews
  - "Ready" (green) - Can be reviewed
  - "Draft" (gray) - Work in progress
  - "Reviewed" (blue) - Already analyzed
- **Metrics row**:
  - Files changed (e.g., "12 files")
  - Lines changed (e.g., "+245 -89")
  - Comments (e.g., "3 comments")
- **Review button**:
  - "Run AI Review" (purple) - Not reviewed
  - "Re-run Review" (outline) - Already reviewed
  - "View Review" (blue) - Has results

#### Card States
- **Hover**: Slight elevation with shadow
- **Loading**: Pulsing animation during review
- **Error**: Red border with error message

### Recent Reviews Section
**What you see**: History of AI reviews below queue

#### Section Header
- **Title**: "Recent AI Reviews"
- **Time filter**: "Today", "This Week", "All Time"
- **Export button**: Download icon

#### Reviews Table
- **Columns**:
  - PR (title + number, clickable)
  - Repository (owner/repo)
  - Review Time (e.g., "2 hours ago")
  - Model Used (e.g., "GPT-4", "Claude")
  - Score (0-100 with color)
  - Findings (count badges)
  - Actions (View, Export, Re-run)

- **Row colors**:
  - Green row: Score > 85
  - Yellow row: Score 70-85
  - Red row: Score < 70

- **Score display**:
  - Number with circular progress
  - Green/yellow/red based on value

- **Findings badges**:
  - Red "3 critical"
  - Yellow "5 warnings"
  - Blue "8 info"

### AI Review Modal
**Opens when**: Click "View Review" or after review completes

#### Modal Header
- **PR title**: Large text
- **Close button**: X in corner
- **Status bar**: Green/yellow/red based on score

#### Review Summary Section
- **Overall Score**: Large number (0-100)
- **Grade**: Letter grade (A+, B, C, etc.)
- **Risk Level**: Low/Medium/High/Critical badge
- **Review Time**: How long analysis took

#### Key Findings Tabs
Tabbed interface with:

1. **Overview Tab**:
   - Summary paragraph
   - Top 3-5 issues
   - Recommendations list

2. **Security Tab**:
   - Vulnerability list with severity
   - Each item shows:
     - Issue title
     - File and line number
     - Description
     - Suggested fix

3. **Performance Tab**:
   - Performance issues found
   - Optimization suggestions
   - Complexity warnings

4. **Quality Tab**:
   - Code style issues
   - Best practice violations
   - Documentation gaps

5. **Tests Tab**:
   - Test coverage analysis
   - Missing test cases
   - Test quality issues
