Add Paragon as an MCP server in Cursor to get code review, security scanning, and analysis tools directly in your editor.

## Prerequisites

- Cursor editor installed
- Node.js 18+
- A Polarity API key from [app.paragon.run](https://app.paragon.run)

## Setup

<Steps>
<Step title="Open MCP settings">

Open Cursor and go to **Settings** → **MCP**. Click **+ Add new MCP server**.

</Step>
<Step title="Configure Paragon">

Select **Type: command** and add the following configuration:

```json
{
  "mcpServers": {
    "paragon": {
      "command": "npx",
      "args": ["-y", "@polarityinc/paragon", "mcp-server"],
      "env": {
        "POLARITY_API_KEY": "your-api-key-here"
      }
    }
  }
}
```

Replace `your-api-key-here` with your Polarity API key.

Alternatively, create a `.cursor/mcp.json` file in your project root with the same configuration.

</Step>
<Step title="Verify the connection">

After saving, you should see **paragon** listed in your MCP servers with a green status indicator.

</Step>
</Steps>

## Available tools

Once connected, Paragon exposes the following tools to Cursor:

| Tool | Description |
|------|-------------|
| **Code review** | Analyze code changes for bugs, security issues, and quality |
| **Security scanning** | Detect vulnerabilities before they reach production |
| **Performance analysis** | Identify memory leaks, inefficient algorithms, and bottlenecks |
| **AI slop detection** | Flag low-quality AI-generated code |
| **Deep review** | Run 8 parallel specialized agents for comprehensive analysis |

## Usage

After setup, Cursor's agent mode can automatically use Paragon tools when relevant. You can also request them directly in chat:

```
Review the changes in this branch for security issues

Run a deep review on the current PR

Check this file for performance problems
```
