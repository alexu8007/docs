Add Paragon as an MCP server in Claude Code to get code review, security scanning, and analysis tools directly in your terminal.

## Prerequisites

- Active Claude subscription (Pro, Max, or API access)
- Node.js 18+
- A Polarity API key from [app.paragon.run](https://app.paragon.run)

## Setup

<Steps>
<Step title="Install Claude Code">

```bash
npm install -g @anthropic-ai/claude-code
```

</Step>
<Step title="Add Paragon as an MCP server">

Run the following command to register Paragon with Claude Code:

```bash
claude mcp add-json paragon '{"command":"npx","args":["-y","@polarityinc/paragon","mcp-server"],"env":{"POLARITY_API_KEY":"your-api-key-here"}}'
```

Replace `your-api-key-here` with your Polarity API key.

</Step>
<Step title="Verify the server is connected">

```bash
claude mcp list
```

You should see `paragon` listed as a configured MCP server.

</Step>
</Steps>

## Available tools

Once connected, Paragon exposes the following tools to Claude Code:

| Tool | Description |
|------|-------------|
| **Code review** | Analyze code changes for bugs, security issues, and quality |
| **Security scanning** | Detect vulnerabilities before they reach production |
| **Performance analysis** | Identify memory leaks, inefficient algorithms, and bottlenecks |
| **AI slop detection** | Flag low-quality AI-generated code |
| **Deep review** | Run 8 parallel specialized agents for comprehensive analysis |

## Usage

After setup, Claude Code can automatically use Paragon tools when relevant. You can also request them directly:

```
> Review the changes in this branch for security issues

> Run a deep review on the current PR

> Check this file for performance problems
```

## Configuration

To update your API key or other environment variables, remove and re-add the server:

```bash
claude mcp remove paragon
claude mcp add-json paragon '{"command":"npx","args":["-y","@polarityinc/paragon","mcp-server"],"env":{"POLARITY_API_KEY":"your-new-api-key"}}'
```
