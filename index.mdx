## Welcome to Polarity

Polarity is a dashboard-only platform — no API required. Connect your repositories and Polarity continuously simulates changes, traces dependencies, flags regressions, and opens high‑quality PRs.

<Note>
  Make sure the Polarity GitHub App is installed for automatic updates.

</Note>

## How it works

<Steps>
<Step title="Install the GitHub App">
  Install the Polarity GitHub App from the dashboard.
</Step>
<Step title="Connect your repos">
  Add orgs and repos from GitHub, GitLab, or self‑hosted. Select which repositories to analyze.
</Step>
<Step title="Choose the model">
  Use Polarity defaults or bring your own. Override per repo or branch.
</Step>
<Step title="Polarity analyzes every commit">
  We simulate behavior, trace dependencies, and flag regressions proactively.
</Step>
<Step title="Polarity publishes PRs">
  We open high‑quality PRs that improve reliability, security, and performance.
</Step>
<Step title="Learns over time — or lock defaults">
  Approvals and comments refine future PRs. Prefer deterministic behavior? Toggle RL off.
</Step>
</Steps>
