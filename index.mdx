## What Polarity Does

- **Automatic Code Analysis**: Scans your repositories continuously for optimization opportunities
- **AI-Generated Pull Requests**: Creates PRs with improvements for performance, security, and code quality
- **Smart PR Organization**: Automatically categorizes and prioritizes pull requests for efficient review
- **Team Collaboration**: Share repositories, track team metrics, and manage permissions
- **Usage Analytics**: Monitor optimization success rates, credit usage, and improvement metrics

## How It Works

<Steps>
<Step title="Connect GitHub">
  Sign in with GitHub and install the Polarity GitHub App
</Step>
<Step title="Add Repositories">
  Select which repositories you want Polarity to monitor and optimize
</Step>
<Step title="Automatic Analysis">
  Polarity 0 continuously analyzes your code and identifies improvements
</Step>
<Step title="Review Pull Requests">
  Review AI-generated PRs and merge the improvements you want
</Step>
</Steps>
